# Fortplotlib
Fortran-native plotting inspired by Python's matplotlib.pyplot and https://github.com/jacobwilliams/pyplot-fortran .

## Requirements
zlib and freetype with development headers.

## Features

### Plot types
- [x] Line
- [ ] Contour
- [ ] Image
- [ ] Surface

### Backends
- [x] PNG
- [x] PDF
- [ ] ASCII
- [ ] Interactive

### Misc
- [ ] Custom line styles
- [ ] Logarithmic scales
- [ ] Secondary axes
- [ ] Math text
